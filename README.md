# 🩺 Disease Prediction Using Machine Learning Algorithms

A Machine Learning-based disease prediction system that predicts possible diseases based on user-selected symptoms. This project uses supervised learning algorithms to improve prediction accuracy and provides an easy-to-use interface for users.

---

## 📌 Project Overview

This project aims to assist users in identifying possible diseases by analyzing symptoms using Machine Learning techniques. The model is trained on a medical dataset containing symptoms and corresponding diseases.

The application helps:
- Predict diseases from symptoms.
- Improve early disease awareness.
- Demonstrate the use of Machine Learning in healthcare.

---

## 🚀 Features

- Disease prediction based on symptoms
- User-friendly Gradio interface
- Data preprocessing and feature selection
- Machine Learning model training
- Performance evaluation using accuracy and confusion matrix
- Easy deployment and testing

## 💻 Installation

Clone the repository

```bash
git clone https://github.com/your-username/Disease-Prediction.git
```

Move into the project directory

```bash
cd Disease-Prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the notebook

```bash
jupyter notebook
```

---

## ▶️ Run the Application

If using Gradio:

```bash
python app.py
```

or run the final cells of the Jupyter Notebook.
